### ☀️ 새롭게 알게된 것

- `tailwindCSS @layer`

마크다운으로 작성한 포스팅 글의 `인라인 코드블럭`이 자꾸 이상해보이는 것이다. 알고보니 내가 별도로 스타일을 지정을 해줘야했다. (기본으로 적용되던 스타일이 있긴하던 것이었는데 안예쁨)

나는 tailwindCSS를 사용하고 있는데, 아래처럼 `@layer`를 이용하면 특정한 요소에 대한 스타일링 규칙을 정의하고, 재사용할 수 있다.

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

@layer base {
  code::before {
    content: "" !important;
  }
  code::after {
    content: "" !important;
  }
  code {
    @apply bg-slate-200 p-1 rounded-sm;
  }
}
```

tailwindCSS는 재사용성 측면에서 단점이 강력한데, @layer를 사용하면 어느정도 해소될 것으로 보인다!
